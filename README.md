<h1>주제 : 소모임 서비스 웹사이트 (MOZIP)</h1><br>
사용언어 : Java / HTML /CSS /JavaScript<br>
개발환경 : SpringBoot<br>
데이터베이스 : Oracle DataBase / dbbeaver<br>
<프로젝트 목표> <br>
① 관리자<br>
② 회원가입 , 로그인<br>
③ 모임 생성, 가입<br>
⑤ 게시글 (등록, 수정, 댓글)<br>
⑥ 마이페이지 (개설 및 가입된 모임)<br>
